Landslide Susceptibility Map

Landslide susceptibility mapping is a key preliminary step in risk mitigation strategies. Susceptibility maps depict the spatial probability levels for a certain area to be prone to landslides, based on local environmental variables


Map generation

To generate the map it was used a model developed in R language that uses the Random Forest ensemble learning method. It is used for classification, regression and other tasks that operate by constructing a multitude of decision trees at training time. For classification tasks, the output of the model is the class selected by most trees.

The environmental factors and the Hazard value have been used as inputs for that process: the model has been fed by training points prepared in earlier steps. At a first glance, the map seemed quite good, because the comparison with the satellite image was pretty coherent.

Validation

After the landslide susceptibility map was produced, a validation process had to be implemented to see accuracy of the outcome:

  2. Only for validation purposes, a simplification has been used and the resulted raster map has been reclassifed into two classes by assigning 0 to the first half of the values and 1 to the second;
  3. The validation tool required as inputs this reclassified layer and, of course, the sampled testing points;
  4. As a result, the script generates an error matrix, also called confusion matrix, that basically reports some statistics on the amount of points that have been correctly or incorrectly classified. The accuracies the matrix computes are statistical indicators that are usually requested for thematic maps;
  • Producer’s accuracy PA of a class is probability that the class present on the ground is also captured by the classification in the thematic raster;
  • User’s accuracy UA of a class shows how often a user of classified map can expect to find the class on the ground;
  • Overall accuracy OA is the proportion of correctly classified pixels out of the total number of pixels.
